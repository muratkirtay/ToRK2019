 74          deviation. 
 75           
 76          @param x: The data to apply to the axis. If multiple values are supplied, 
 77          the largest bounds are used.

 79          @param multiplier: The multiplier to scale by. Recommended is 6 for Y axis

 84                  min_v = np.min(v) 
 85                  max_v = np.max(v) 
 86                  std = multiplier / 5. if min_v == max_v else np.std(v) 
 87                  lb = 0 if min_v > 0 else min_v - std 
 88                  ub = max_v + std 
 89                  return lb, ub
