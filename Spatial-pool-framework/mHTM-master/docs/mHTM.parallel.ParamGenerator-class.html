Class ParamGenerator

source code

Build parameters properly bounded for the SP. Item generation follows scikit-learn's ParameterSampler; therefore, the order is guaranteed.

Assumes that at least the following parameters are being varied: ncolumns, nactive, nsynapses, seg_th, and log_dir.
