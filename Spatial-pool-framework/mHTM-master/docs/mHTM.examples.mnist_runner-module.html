full_mnist(base_dir, new_dir, auto_update=False)

source code 

Execute a full MNIST run using the parameters specified by ix.

Parameters:
  • base_dir - The full path to the base directory. This directory should contain the config.
  • new_dir - The full path of where the data should be saved.
  • auto_update - If True the permanence increment and decrement amounts will automatically be computed by the runner. If False, the ones specified in the config file will be used.
