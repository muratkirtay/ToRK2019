main(ntrain=800, ntest=200, niter=10, nsplits=5, global_inhibition=True, seed=None)

source code 

Build the information needed to perform CV on a subset of the MNIST dataset.

Parameters:
  • ntrain - The number of training samples to use.
  • ntest - The number of testing samples to use.
  • niter - The number of parameter iterations to use.
  • nsplits - The number of splits of the data to use.
  • global_inhibition - If True use global inhibition; otherwise, use local inhibition.
  • seed - The seed for the random number generators.
Returns:
The full set of X, the full set of Y, the keyword arguments for the classifier, the params for CV, and the CV.

main_slurm(log_dir, ntrain=800, ntest=200, niter=5, nsplits=3, global_inhibition=True, partition_name='debug', seed=None)

source code 

Perform CV on a subset of the MNIST dataset, using SLRUM. Iterations will be run in complete parallel. Splits within an iteration will be run sequentially.

Parameters:
  • log_dir - The directory to store the results in.
  • ntrain - The number of training samples to use.
  • ntest - The number of testing samples to use.
  • niter - The number of parameter iterations to use.
  • nsplits - The number of splits of the data to use.
  • global_inhibition - If True use global inhibition; otherwise, use local inhibition.
  • partition_name - The partition name of the cluster to use.
  • seed - The seed for the random number generators.
